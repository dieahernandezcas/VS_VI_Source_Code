Analysis of Selective and Indiscriminate Violence in Colombia

See Repository in GitHub

The experiment designed to test the proposed hypothesis (Figure 1) uses official data on Colombia’s armed conflict from 1958 to 2021. Monthly time series were created for six variables: cases and victims across three dimensions of violence—collective, selective, and indiscriminate. Each series was normalized into dynamics of 1, 0, and -1 for both escalation (based on change from the previous month) and intensity (relative to the average of the previous three years). To forecast future values, three sliding window models are used to estimate the probability of escalation and intensity at time t+1 based on patterns observed at t-1. These models compare the state at t-1 with historical sequences t_n to calculate the likelihood of each outcome. Forecasts are evaluated both independently and jointly for escalation and intensity, and their performance is compared against a traditional weighted moving average model using a sliding window. Model accuracy is assessed using the weighted F1 score, following the approach of Perry (2010).

About the Project

This project focuses on the quantitative analysis of Selective Violence (VS) and Indiscriminate Violence (VI) in Colombia. Using detailed data on violent events (Cases) and affected persons (Victims), we aim to understand the dynamics of violence at the national, departmental, and regional levels.

The analysis includes data cleaning and processing, and the determination of key metrics such as **Escalation** (comparison with the previous month) and **Intensity** (comparison with the weighted average of the last 3 years) for both Cases and Victims.

Algorithms Used

The predictive algorithms implemented are SEPP, KDE, and Naive KDE. These models were implemented using base code developed by the QuantCrime Lab at the University of Leeds, available at: QuantCrime Lab - PredictCode.

Project Structure

The project is organized into key folders for experimental scripts, custom libraries (including base code from University of Leeds), examples, and data (raw, processed, results for Bogotá and Chicago). A global configuration file manages variables across scripts.

For detailed step-by-step instructions, please see the installation guide in the README.md.

Key Metrics

The project calculates the following metrics for Cases and Victims at the Country, Department, and Region levels:

  • **Escalation:** Change in the number of Cases or Victims compared to the immediately preceding month.
  • **Intensity:** Comparison of the current number of Cases or Victims with the weighted average over the last 3 years.

Exact formulas are detailed in the README.md.
