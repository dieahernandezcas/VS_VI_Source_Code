Análisis de Violencia Selectiva e Indiscriminada en Colombia

Ver Repositorio en GitHub

Este proyecto se enfoca en el análisis cuantitativo de la Violencia Selectiva (VS) y la Violencia Indiscriminada (VI) en Colombia. Utilizando datos detallados sobre eventos violentos (Casos) y personas afectadas (Víctimas), buscamos comprender las dinámicas de la violencia a nivel nacional, departamental y regional.

El análisis incluye la limpieza y procesamiento de datos, y la determinación de métricas clave como la **Escalada** (comparación con el mes anterior) y la **Intensidad** (comparación con el promedio ponderado de los últimos 3 años) para Casos y Víctimas.

Sobre el Proyecto

Este repositorio contiene el código fuente y los scripts utilizados para el análisis de la Violencia Selectiva (VS) y la Violencia Indiscriminada (VI) en Colombia. El objetivo es proporcionar herramientas para el procesamiento de datos y el cálculo de métricas que permitan monitorear y entender la evolución de estos tipos de violencia en diferentes niveles geográficos del país.

Instalación

Para configurar el proyecto localmente, asegúrate de tener Python 3.7+, pip y git instalados. Luego, clona el repositorio, crea un entorno virtual e instala las dependencias listadas en requirements.txt.

Consulta la sección de instalación en el README.md para instrucciones detalladas paso a paso.

Métricas Clave

El proyecto calcula las siguientes métricas para Casos y Víctimas a nivel País, Departamento y Región:

  • **Escalada:** Cambio en el número de Casos o Víctimas respecto al mes inmediatamente anterior.
  • **Intensidad:** Comparación del número actual de Casos o Víctimas con el promedio ponderado de los últimos 3 años.

Las fórmulas exactas se detallan en el README.md.

Resultados

Los resultados del procesamiento de datos y el cálculo de métricas se almacenan en la carpeta results/. Estos incluyen tablas resumen y, potencialmente, visualizaciones generadas por los scripts o notebooks.
