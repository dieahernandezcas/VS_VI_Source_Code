Analysis of Selective and Indiscriminate Violence in Colombia

View Repository on GitHub

This project focuses on the quantitative analysis of Selective Violence (VS) and Indiscriminate Violence (VI) in Colombia. Using detailed data on violent events (Cases) and affected persons (Victims), we aim to understand the dynamics of violence at the national, departmental, and regional levels.

The analysis includes data cleaning and processing, and the determination of key metrics such as **Escalation** (comparison with the previous month) and **Intensity** (comparison with the weighted average of the last 3 years) for both Cases and Victims.

About the Project

This repository contains the source code and scripts used for the analysis of Selective Violence (VS) and Indiscriminate Violence (VI) in Colombia. The objective is to provide tools for data processing and metric calculation that allow for monitoring and understanding the evolution of these types of violence across different geographical levels of the country.

To set up the project locally, ensure you have Python 3.7+, pip, and git installed. Then, clone the repository, create a virtual environment, and install the dependencies listed in requirements.txt.

Consult the installation section in the README.md for detailed step-by-step instructions.

Key Metrics

The project calculates the following metrics for Cases and Victims at the Country, Department, and Region levels:

  • **Escalation:** Change in the number of Cases or Victims compared to the immediately preceding month.
  • **Intensity:** Comparison of the current number of Cases or Victims with the weighted average over the last 3 years.

Exact formulas are detailed in the README.md.

Results

Results from data processing and metric calculation are stored in the results/ folder. These include summary tables and potentially visualizations generated by the scripts or notebooks.
